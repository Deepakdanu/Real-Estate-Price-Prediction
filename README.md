
# 🏠 Real Estate Price Prediction

A Machine Learning project that predicts real estate property prices using various property features such as area, number of bedrooms, bathrooms, location, and other attributes.

## 📌 Project Overview

This project uses machine learning algorithms to estimate house prices based on historical real estate data. The model helps buyers, sellers, and real estate professionals make informed pricing decisions.

## ✨ Features

- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Machine Learning Model Training
- Model Evaluation
- Price Prediction

## 🛠 Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook

## 📂 Dataset

The dataset contains property information including:

- Area
- Bedrooms
- Bathrooms
- Location
- Price

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/Deepakdanu/Real-Estate-Price-Prediction.git
```

Install dependencies:

```bash
pip install -r requirements.txt
```

## ▶️ Run the Project

Open Jupyter Notebook:

```bash
jupyter notebook
```

Open:

```
Real Estate.ipynb
```

## 📊 Model Performance

The project compares multiple machine learning models and selects the best-performing model based on evaluation metrics.
